# TradeVault (static UI)

Simple static trading journal UI. You can run it locally using Node or Python.

Run with Node (recommended if Node is installed):

1. Install Node (if needed).
 2. Install project dependencies and start the server:
 
```
npm install
Open http://localhost:3000 in your browser.

Run with Python (no install required):

```
# from project folder
python -m http.server 3000
```
